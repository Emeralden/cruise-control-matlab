### Cruise Control System



This project simulates a basic cruise control system using MATLAB and models the vehicle dynamics using a first-order LTI ODE. A PI controller is implemented to achieve desired speed tracking with zero steady-state error.



##### Features

\- Open-loop and closed-loop analysis

\- PI controller tuning

\- Root locus plots for Kp and Ki

\- MATLAB ODE simulation and plotting



##### Results

\- Quick rise time

\- Zero steady-state error

\- Stability guaranteed via Routh-Hurwitz and root locus



##### Tech Used

\- MATLAB

\- Control Systems

\- ODE Solver (`ode45`)

\- Root Locus
